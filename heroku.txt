-- Deployment Heroku --

Create new app di heroku
Kemudian ketik command berikut:
- git clone <link github>
- heroku login
- heroku git:remote -a <nama app>
- git add .
- git commit -am <"beri komentar">
- git push heroku <nama branch>
Cek web app yang sudah dibuat melalui <nama app>.herokuapp.com
